# 方向06：数据处理及统计综合作业

## 题目描述
构建一个AI驱动的大规模数据分析代理，支持从原始数据到洞见报告等全自动化流程中的任一功能。

## 题目要求
- **数据准备**：选择大规模公开数据集（如Kaggle竞赛、Common Crawl子集）。
- **模型选择**：基于LLM Agent（如LangGraph、DSPy）实现数据清洗、EDA、建模、报告生成。
- **核心功能**：自动清洗、特征工程、可视化、统计检验、洞见总结。
- **迭代优化**：更新prompt链、加入工具（Pandas、Statsmodels），提升报告质量。
- **部署**：上传CSV后自动输出Markdown/PDF报告。
- **评估**：人工评分报告准确性与洞见深度。

## 最后提交要求
1. **代码提交**  
   - 独立公开仓库，支持上传数据自动分析。

2. **报告提交**  
   - 附至少一份完整自动化生成的分析报告作为示例。

3. **文件名命名规范**  
   - 报告文件：`学号-姓名-06-Data-Processing.md` 或 `学号-姓名-06-Data-Processing.pdf`
