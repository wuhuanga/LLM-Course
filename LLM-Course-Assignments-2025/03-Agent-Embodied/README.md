# 方向03：智能体及具身智能综合作业

## 题目描述
构建一个基于LLM的具身智能体（Embodied Agent），在模拟环境中执行复杂任务（如家庭助手、物体操作或导航）。

## 题目要求
- **数据准备**：使用公开具身数据集（如VirtualHome、AI2-THOR或Minecraft），或合成轨迹数据。
- **模型选择**：基于VLA模型（如OpenVLA、RT-2）或LLM+工具调用（如ReAct），实现感知-规划-行动闭环。
- **核心功能**：多步规划、工具使用（抓取、移动）、错误恢复。
- **迭代优化**：可微调行动策略、加入记忆模块，提升任务成功率。
- **部署**：在模拟器（如Habitat、CARLA）中运行Demo，录制视频。
- **评估**：任务成功率、步骤效率、泛化能力。
